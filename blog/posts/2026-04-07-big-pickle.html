Big Pickle: OpenCode's Stealth Coding Model

Big Pickle is OpenCode Zen's flagship model, and it's something a bit unusual in the AI world - a "stealth model." That means OpenCode hasn't publicized exactly which base model it uses, but they've optimized it specifically for coding agent workflows.

What Is a "Stealth Model"?

Unlike companies that clearly state "we use Claude" or "we use GPT," Big Pickle's architecture isn't publicly disclosed. This is actually common in the AI space - companies optimize base models and customize them for specific use cases.

What matters isn't the underlying architecture - what matters is how well it performs for coding tasks. And in my experience, Big Pickle performs well.

The Specifications We Know

  • Provider: OpenCode Zen (opencode.ai)
  • Model ID: opencode/big-pickle
  • Pricing: Free (during beta)
  • Context: 200,000+ tokens

What It Can Do

Big Pickle has direct access to OpenCode's full toolset:

  • File operations: Read, write, edit files
  • Terminal commands: Run bash commands
  • Web search: Search and fetch web content
  • Task management: Launch sub-agents
  • Code execution: Run and debug code

My Experience

I've been using Big Pickle to build this website and the Obsidian vault that tracks my learning. Here's what stands out:

  • Fast: Responses are snappy, no long waits
  • Reliable: Consistently produces working code
  • Conversational: Easy to chat with and direct
  • Tool-aware: Knows what tools are available

How It Compares to Other Free Models

OpenCode Zen offers several free models:

  • Big Pickle: Optimized for coding agents, currently my choice
  • Qwen3.6 Plus Free: 1M context, excellent for long files
  • Nemotron 3 Super Free: NVIDIA's open weights model
  • MiniMax M2.5 Free: Office productivity focus

While Big Pickle may not have the longest context window (Qwen wins there), it's optimized for the terminal workflow that OpenCode provides.

The Free Beta

Big Pickle is currently free during OpenCode Zen's beta period. There's an important note: data collected during this period may be used to improve the model.

This is fair - you're getting something for free, and OpenRouter gets real-world usage data. Just know that your conversations with it may be logged for model improvements.

Should You Use Big Pickle?

If you're using OpenCode and want a model that "just works" for coding tasks, Big Pickle is an excellent choice. It's:

  • Pre-configured with OpenCode
  • Free during beta
  • Optimized for the terminal workflow
  • Reliable and fast

The stealth model approach might not appeal to everyone - some prefer knowing exactly what model they're using. But if you care about performance over transparency, Big Pickle delivers.

For my 90-Day Pivot Challenge learning journey, Big Pickle has been an invaluable tool. It's helping me build, research, and learn without any cost barriers.
