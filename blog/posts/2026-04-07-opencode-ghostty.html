Using OpenCode in Ghostty Terminal

Categories: Using AI, Free Tools

OpenCode is an AI coding assistant that runs in your terminal. When you pair it with Ghostty (a great terminal emulator for Mac), you get a powerful development setup.

Why This Combo Works

  • Speed: Ghostty is incredibly fast
  • AI access: OpenCode brings AI assistance to your workflow
  • Privacy-first: Your data stays between you and the AI
  • Free models: You can use free models via OpenRouter

The Setup

Here's what you need to know:

  1. Ghostty: Download from ghostty.org (Mac only, but fast)
  2. OpenCode: Install via brew install opencode
  3. Model: Connect to OpenRouter for free models

What OpenCode Can Do

  • Read and edit code in your editor
  • Run terminal commands
  • Search the web for information
  • Create and manage files
  • Debug and explain code

My Experience

Using OpenCode in Ghostty has become my primary development setup. The terminal-based workflow feels natural, and having AI assistance available without leaving my workflow is valuable.

The free models via OpenRouter mean I can experiment and learn without worrying about costs. It's perfect for someone like me who's pivoting at 50+ and learning new tools.
