Ghostty Terminal: A Fast, Modern Terminal for Mac

Categories: Free Tools, Other

Ghostty is a terminal emulator for Mac that focuses on speed, functionality, and a great user experience. It's developed by mitchellh (known for HashiCorp tools like Vagrant and Packer).

Key Features

  • Ridiculously fast: Built with Zig, uses GPU rendering
  • Native Feel: Feels like a first-party Mac app
  • Tabs and splits: Great window management
  • Deep links: ghostty:// URL scheme for integrations
  • Configurable: Plain text configuration file

Installation

The easiest way:

brew install ghostty

Or download from ghostty.org

Why I Like It

Coming from iTerm2, Ghostty feels like a breath of fresh air:

  • Startup: Immediately ready, no waiting
  • Scrolling: buttery smooth
  • Memory: Light on resources
  • Look: Clean, modern appearance out of the box

The Trade-off

It's Mac-only. If you need cross-platform, you'll need another option. But if you're on Mac and want the best terminal experience, Ghostty is excellent.

It's free and open source. The developer is clearly building something he wants to use, and it shows in the quality.
